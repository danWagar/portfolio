PROJECTS

ManageLazily is a project management app that allows business owners to manage their business as a hierarchical structure. The owner/admin can promote workers to project managers and then create and assign projects to them. Managers then create tasks needed to complete the project and assign employees to them. Employees log hours, request edits to a task, and submit tasks for approval by the project manager who either approves it or requests revisions.

The app was created by myself and 4 other developers where I served as the UI/UX design lead and contributed to the code base. The design process included the initial design layout in Figma which would serve to guide the entire front end development process. The design, styling and images are all my original work.

For players of the game Borderlands 3 it is all about the loot. Unfotunately the game does not provide many in-game resources for keeping track of loot. The Borderlands 3 Gear Tracker web app was designed as a resource such that players will always be able to know what loot is in their inventory and where it is located.

This RESTful app was developed with React on the client side and with Node, Express, PostgreSQL and Knex on the server side with testing handled by Chai and Mocha.

As with all my projects there is a strong emphasis on design. The app displays large amounts of data and the layout is designed to be both organized and dynamically expandable and collapsible.

Le Beguette was a one week paired programming project where, given a front end and back end framework, we were asked to implement a spaced repetition learning app using a linked list algorithm on the back end where the lists nodes are persistently stored in the database. We also implemented the front end React components related to the spaced repetition learning process.

For this project I was responsible for all back end code, including the linked list implementations, while also having some contributions on the front end code. I was also responsible for design and all styling as well as ensuring tests passed on both the front and back end.

The My Bookmarks App is an exercise in learning to implement a RESTful Web API and is written in pure Javascript as a precursor to learning React.

Like the quiz app below it follows the model-view-controller design paradigm, but in this case a object oriented design was implemented. It expands upon the lessons learned in developing the quiz app to ensure that the state of the front end data store remains in-line with the data stored on the backend server.

A simple and clean design makes for an enjoyable and intuitive user experience.

The NFL Records Quiz app was developed as an exercise in producing a single page web app with dynamically generated html for all event transitions.

The app follows the model-view-controller design paradigm. It utilizes a store to track the users performance on the quiz as well as to display questions, whether a user correctly answered a question, and final results for the quiz.

As I highly value the power of good design, much focus was given to ensuring that the app was attractive and engaging. User feedback is clear and presented in a fun way by incorporating images and a final results ranking.
